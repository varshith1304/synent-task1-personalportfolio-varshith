## 👤 About

> Personal Portfolio Website — a clean, single-page site built with **HTML & CSS** to showcase profile, skills, and contact details.

This portfolio belongs to **M Varshith Reddy**, a B.Tech student and aspiring Full Stack Developer who is eager to learn new technologies and enjoys solving problems.

## 🚀 How to Use

```bash
# 1. Download the file
portfolio.html

# 2. Open it in your browser
double-click portfolio.html

# 3. Edit your real details
Update the email, phone, LinkedIn, and GitHub links
```

<br/>

## 🎨 Customization Guide

| Section | What to Edit |
|---|---|
| 🪪 Header | Name and title text |
| 📝 About Me | Paragraph content |
| 🛠️ Skills | Add/remove `<span class="skill-tag">` items |
| 📬 Contact | Replace placeholder links with real email/phone/social URLs |

<br/>

## 📌 Notes

- ⚡ No external dependencies or frameworks — pure HTML & CSS
- 🎨 Color theme and fonts are easily changed via the CSS variables at the top of the `<style>` block
- 📱 Fully responsive with media queries for smaller screens
